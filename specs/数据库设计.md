# 数据库表设计文档

## 1. 管理员表 (Admin) - 使用现有 User 表

### 表结构
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|
| _id | ObjectId | 是 | 主键 |
| name | String | 是 | 管理员姓名 |
| username | String | 否 | 登录用户名 |
| phone | String | 是 | 手机号（登录账号） |
| password | String | 是 | 密码（加密存储） |
| email | String | 否 | 邮箱 |
| role | String | 是 | 角色：admin, super_admin |
| avatar | String | 否 | 头像 |
| department | String | 否 | 部门 |
| position | String | 否 | 职位 |
| status | String | 是 | 状态：active, inactive |
| lastLoginAt | Date | 否 | 最后登录时间 |
| lastLoginIp | String | 否 | 最后登录IP |
| permissions | [ObjectId] | 否 | 权限ID列表 |
| createdAt | Date | 是 | 创建时间 |
| updatedAt | Date | 是 | 更新时间 |

## 数据库关系图

```
Admin (管理员)
  ├───> Role (角色)
  │       └──> Permission (权限)
  │       └──> Menu (菜单)
  │
  ├──> Member (会员)
  │       └──> RedPacket (红包)
  │
  ├──> Course (课程)
  │       └──> ProjectVideo (项目视频关联)
  │               └──> Video (视频)
  │               └──> Project (项目)
  │
  └──> AuditLog (审计日志)
```
