# CRM 系统项目结构（UniApp + Vue3 版本）

## 项目概述

基于 **UniApp + Vue3** 构建的三合一客户关系管理（CRM）系统，一套代码同时支持：
- ✅ 微信小程序
- ✅ Android/iOS APP
- ✅ H5 网页（手机 + PC）

## 技术栈

- **前端框架**: UniApp + Vue 3 (Composition API)
- **构建工具**: Vite
- **UI 组件库**: uView Plus
- **状态管理**: Pinia
- **后端框架**: Node.js + Express（保持不变）
- **数据库**: MongoDB（保持不变）

## 与旧架构对比

| 旧架构 (React Native) | 新架构 (UniApp + Vue3) |
|----------------------|------------------------|
| src/screens/ | src/pages/ |
| src/redux/slices/ | src/stores/ |
| Redux Toolkit | Pinia |
| React Navigation | UniApp 路由 |
| axios | uni.request |

## 构建命令

```bash
npm run dev:h5              # 开发 H5
npm run dev:mp-weixin       # 开发微信小程序
npm run build:h5            # 构建 H5
npm run build:mp-weixin     # 构建小程序
```

> 后端 API 接口完全兼容，无需修改。
