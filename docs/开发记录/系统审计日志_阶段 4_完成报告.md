#### GET /api/audit-logs
**功能**: 查询审计日志列表（支持多条件筛选+分页）

**请求参数**:
```javascript
{
  startTime: '2026-04-01T00:00:00.000Z',   // 可选，开始时间
  endTime: '2026-04-13T23:59:59.999Z',     // 可选，结束时间
  userId: 'ObjectId',                     // 可选，用户ID过滤
  actionType: 'login',                    // 可选，操作类型模糊匹配
  resource: 'video',                      // 可选，资源类型
  success: 'true',                        // 可选，成功/失败
  keyword: 'error',                       // 可选，关键词搜索
  page: 1,                                // 可选，页码（默认1）
  limit: 20                               // 可选，每页数量（默认20，最大100）
}
```

**响应格式**:
```javascript
{
  success: true,
  data: {
    logs: [...],                           // 日志数组
    pagination: {
      total: 1000,                         // 总记录数
      page: 1,                             // 当前页
      limit: 20,                           // 每页数量
      totalPages: 50                       // 总页数
    }
  }
}
```

**特性**:
- ✅ 支持时间范围查询
- ✅ 支持多维度筛选（用户、类型、资源、状态）
- ✅ 支持关键词模糊搜索（action/resource/errorMessage）
- ✅ 分页功能完整
- ✅ 按时间倒序排列

## 技术亮点

### 1️⃣ 高性能聚合查询

统计 API 使用 MongoDB Aggregation Pipeline：

```javascript
const last7DaysStats = await AuditLog.aggregate([
  { $match: { timestamp: { $gte: sevenDaysAgo } } },
  { 
    $group: {
      _id: { date: "$timestamp", isSuccess: "$success" },
      count: { $sum: 1 }
    }
  },
  {
    $group: {
      _id: "$_id.date",
      successCount: { $sum: { $cond: [{ $eq: ["$_id.isSuccess", true] }, "$count", 0] } },
      failCount: { $sum: { $cond: [{ $eq: ["$_id.isSuccess", false] }, "$count", 0] } }
    }
  }
]);
```

**优势**:
- 单次数据库查询获取所有统计数据
- 服务端高效聚合，减少网络传输
- 支持大数据量（百万级日志）

### 2️⃣ 智能 CSV 导出

```javascript
const BOM = '\uFEFF';  // UTF-8 BOM，确保 Excel 正确显示中文
res.send(BOM + csv);
```

**特性**:
- Excel 兼容（中文不乱码）
- 自动转义特殊字符
- 可定制字段列表

### 3️⃣ 安全的清理机制

```javascript
const sevenDaysAgo = new Date();
sevenDaysAgo.setDate(sevenDaysAgo.getDate() - 7);

if (before < sevenDaysAgo) {
  return res.status(400).json({
    success: false,
    message: '至少保留最近 7 天的日志'
  });
}
```

**保障**:
- 强制保留最近7天数据
- 清理操作自身也被记录
- 防止误删重要日志

### 4️⃣ 全面的查询支持

支持 **8 种筛选条件**:
1. 时间范围（startTime / endTime）
2. 用户ID（userId）
3. 操作类型（actionType，模糊匹配）
4. 资源类型（resource）
5. 成功/失败（success）
6. 关键词搜索（keyword，跨字段）
7. 分页（page + limit）
8. 排序（固定按时间倒序）
