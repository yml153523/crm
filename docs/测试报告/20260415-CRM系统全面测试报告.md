## 🐛 发现并修复的问题（本次部署）

### 问题 1：后端缺少 adminUserController.js
- **严重程度**: P1 - 致命
- **现象**: PM2 启动报错 `Cannot find module '../controllers/adminUserController'`
- **根因**: 文件不存在但被 adminUser.js 引用
- **修复**: 创建完整的 adminUserController.js（6个方法）
- **状态**: ✅ 已修复

### 问题 2：后端缺少 bcrypt 模块
- **严重程度**: P1 - 致命
- **现象**: PM2 启动报错 `Cannot find module 'bcrypt'`
- **根因**: user.js 引用了 bcrypt 但未安装
- **修复**: `npm install bcrypt --save`
- **状态**: ✅ 已修复

### 问题 3：PM2 端口占用导致旧代码运行
- **严重程度**: P2 - 高
- **现象**: POST /api/users 返回 "Cannot POST"
- **根因**: PM2 restart 后旧进程未完全释放端口5011
- **修复**: `pm2 delete` → `fuser -k 5011/tcp` → `pm2 start`
- **状态**: ✅ 已修复
