## 关键修复和改进

### 1. 视频 URL 端口问题
**问题**: 数据库中记录的视频 URL 使用 80 端口而不是 5011 端口  
**解决**: 测试脚本中增加 URL 端口替换逻辑，确保正确访问

### 2. 管理员登录 API
**问题**: 测试脚本使用错误的登录 API 路径  
**解决**: 更正为 `/api/auth/admin-login`，使用 `username` 字段而非`phone`

### 3. 会员管理路由
**问题**: 不存在 `/api/members` 路由  
**解决**: 使用 `/api/users` 路由进行会员管理

### 4. 静态资源部署
**问题**: 之前的部署脚本跳过 `node_modules` 目录，导致字体和图标文件缺失  
**解决**: 修改部署脚本，允许上传 `assets/node_modules` 中的资源文件
