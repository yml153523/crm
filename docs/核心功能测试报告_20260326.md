## 🚨 严重问题

### 问题1: 题目创建返回500错误 ❌

**严重程度**: 高

**测试过程**:
```
POST /api/questions
请求: {
  "video": "69c3d97b8c7cd25e904cf928",
  "type": "single",
  "question": "测试题目 - 今天的日期是2026年吗?",
  "options": [{"key": "A", "content": "是的"}, {"key": "B", "content": "不是"}],
  "correctAnswer": "A",
  ...
}
响应: 500 Internal Server Error
```

**根本原因**: 数据模型定义不匹配

```javascript
// Question 模型定义
correctAnswer: {
  type: [Number],  // ❌ 定义为数字数组
  required: true
}

// 但前端/测试传的是字符串 "A"
correctAnswer: "A"  // ❌ 类型不匹配
```

**正确格式应该是**:
```javascript
correctAnswer: [1]  // 数字索引，表示第二个选项（0=A, 1=B, 2=C, 3=D）
```
